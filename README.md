# sayitout  
An anonymous confession website, built using Node.js and mongoDB  

Hosted [here](https://www.sayitout.tk/)

## TO run Locally  
```git clone git@github.com:husain3012/sayitout.git```  
```cd sayitout```  
Add envirioment vairable ```DATABASE=<mongodb URI>```  
```npm i```  
```node server.js```
